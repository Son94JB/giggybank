## **개발동기**

><h2>거지방 확장과 과소비 문화 축소를 위한 앱</h2>

    - 거지방의 채팅기능을 커뮤니티로 확장
    - 유니티로 개발된 게임을 통한 경쟁심 자극
    - 매 주 업데이트 되는 랭킹 시스템
    - 소비 패턴 분석

    

> ## 팀원 소개

    - 팀장 : 김준성(Front-end)
    - 팀원 : 손준배(Back-end)
    - 팀원 : 이현근(Back-end)
    - 팀원 : 김재욱(Back-end)
    - 팀원 : 김현지(Back-end)


>## **사용된 기술**
Frontend<br>
클린 아키텍처, RETROFIT, 멀티 모듈, 카카오 API, HILT, FIREBASE

Backend<br>
MVC패턴, REDIS 서버 사용, FIREBASE, 스케줄러, S3, 인터셉터, FasTexT, Gunicorn

Infra<br>
Docker, Jenkins, EC2, NGINX
